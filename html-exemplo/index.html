Olá Git/Github!

Git é uma ferramenta de versionamento, cria um repositório local

Github cria um repositório remoto, serve também como uma 'rede social' de programadores

Commit é o ato de 'salvar' no repositório as alterações feitas no código

De acordo com o Cambridge a palavra commit é fazer algo ruim ou ilegal, ou ainda tomar uma decisão definitiva a respeito de algo
